# webxr-suika

Fan-made AR version of the popular fruit-matching game with hand tracking. Built with Three.js, Rapier.js, and GSAP. It runs on any XR device with a WebXR-capable browser, and has been tested on Spectacles, Quest 3, and Vision Pro. <br> The project is a small experiment exploring WebXR capabilities and is not affiliated with any company.

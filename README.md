# webxr-ar-suika

![Preview](preview.gif)

Fan-made AR version of the popular fruit-matching game with hand tracking. Built with Three.js, Rapier.js, and GSAP. Optimized for true AR glasses, but runs on any XR device with a WebXR-capable browser. The project is a small experiment exploring WebXR capabilities and is not affiliated with any company.

**Live demo:** https://webxr-suika.vercel.app/

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/dmvrg/webxr-suika.git
cd webxr-suika
npm install
npm run dev
```
